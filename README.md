# Project Title

Apple store reviews from ITunes RSS feed

Backend: https://github.com/mina-kleid/apple-store-reviews

## Getting Started

### Prerequisites

- Node v18.15.0

### Installation

```bash
git clone https://github.com/mina-kleid/apple-store-reviews-frontend
cd apple-store-reviews-frontend
npm install
```

## Usage

```bash

# Start the app
npm run start

```
